# Table Project

A responsive HTML table with sidebar navigation and top header bar.

## Features

- **50×50 Data Grid**: Table with 50 rows and 50 columns
- **Consistent Numbering**: All rows follow the same pattern (1-50)
- **Sticky Navigation**: Top header bar that stays fixed
- **Sidebar Navigation**: Fixed sidebar with menu, filters, and stats
- **Responsive Design**: Scrollable table within main content area
- **Modern UI**: Clean design using Tailwind CSS

## Structure

- **Top Navigation**: Fixed header with action buttons
- **Sidebar**: Fixed navigation with filters and quick stats
- **Main Content**: Scrollable table container

## Table Details

- **Dimensions**: 50 rows × 50 columns = 2,500 total cells
- **Pattern**: Each row contains numbers 1 through 50
- **Styling**: Clean, minimal design with borders and hover effects

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Usage

Simply open `index.html` in a web browser to view the table with navigation.
